## 平行组件之间的通信
需要借助一个空的vue对象。var Bridge = new Vue();（定义一个公共的js文件）发送依旧是用Bridge.$emit('自己定义的事件','需要发送的数据')，接收的时候，需要在钩子函数mounted中接收，用Bridge.$on('相对应的事件',回调函数function(msg){});

```
import Vue from 'vue'
export default new Vue()

在mounted函数中写：
Bridge.$on('refreshHomeworkAnswerList' ,(data) => {
});
```
原理：
子 =>父 父=>子（在父组件那里周转一下）
